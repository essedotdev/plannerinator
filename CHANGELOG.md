# Changelog

All notable changes to Plannerinator will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial project setup with Next.js 15 and Better Auth
- Database schema foundation with Drizzle ORM
- Authentication system (login, register, password reset, email verification)
- RBAC system with user and admin roles
- Dashboard layout with responsive sidebar navigation

## [0.1.0] - 2025-01-21

### Added
- Initial release of Plannerinator
- Core authentication with Better Auth
- PostgreSQL database with Drizzle ORM
- Type-safe schema with user roles
- Protected dashboard routes
- Profile management
- User management (admin only)
- Dark mode support
- Responsive layout with shadcn/ui components
